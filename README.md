# test3

Репозиторий с решениями задач.

## Структура

| Директория | Описание |
|------------|----------|
| [email-domain-check](./email-domain-check/) | Проверка email-доменов по MX-записям |
| [telegram-integration](./telegram-integration/) | Отправка текста из .txt в приватный Telegram-чат через бота. [Скриншот результата](./telegram-integration/example.png) |
| [architecture](./architecture/) | Архитектура email-аутрича на 1200 адресов (Yandex Cloud, мультитенантность, ~8 400 ₽/мес) |

## Требования

- Python 3.10+
- Windows

## Блиц: AI-стек

- **IDE и плагины:** Cursor; ChatGPT — для первичной формулировки и структуризации ТЗ
- **Модель и задачи:** для кода — модели с сильным reasoning; для быстрых правок — более быстрые. Основные задачи: Telegram-боты под бизнес-сценарии, скрипты автоматизации, архитектурные решения.
- **MCP:** web fetch (чтение документации, страниц), task (делегирование сложных многошаговых задач подагентам)
- **Cursor rules:** в `.cursor/rules/` задаю: модульность кода, секреты в `.env`, единый стиль (Black/ruff), docstrings для публичного API
